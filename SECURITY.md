# Security Policy

## Supported Versions

Below are the versions of Aura Music that receive security updates:

| Version | Status         | Description                                                    |
| ------- | -------------- | -------------------------------------------------------------- |
| v1.0.0  | ✅ Supported   | **Aura Music (Desktop Beta) Latest** - Current stable release. |
| < 1.0   | ❌ End of Life | Early development builds.                                      |

## Reporting a Vulnerability

We take the security of Aura Music seriously. If you discover a security vulnerability, please follow these steps:

1.  **Do not** create a publicly visible GitHub issue.
2.  Please report vulnerabilities via the "Private vulnerability reporting" feature on this repository (if enabled) or contact the maintainer directly.
3.  We aim to acknowledge reports within 48 hours and will provide a fix in the next patch release.

Thank you for helping keep Aura Music safe!
